Connect Amazon Seller Central

Pull BSR rank, ad spend, ACOS, and sales data directly into this dashboard via Make.

01 Go to Amazon Seller Central → Apps & Services → Manage Your Apps → Authorize a new app (SP-API)
02 In Make, add an HTTP module hitting https://sellingpartnerapi-na.amazon.com/reports/2021-06-30/reports with your LWA refresh token
03 Request report type GET_SALES_AND_TRAFFIC_REPORT — upsert into a new amazon_daily Supabase table
04 For Sponsored Products ACOS, use the Advertising API: advertising.amazon.com/API/v2/reports
05 Once the table is populated, this page will show BSR, units sold, ACOS, TACoS, and ad spend automatically
